Formula1 DataBucket

This project was inspired by my passion for Formula 1. It uses get requests to the Ergast API to retrieve fastest laps for tracks and Driver/Team information. Please be patient as the API is very slow to respond at times :-P

The only page reloading that happens is when you click on a nav button at the top. Everything else is done via AJAX.

The graphs that are displayed on the driver and team pages are created by a custom graphing library that I built from scratch.

Updated 11/12/15 at 4:03pm --> AJAX requests are now more reliable, added indeterminate progress indicators while AJAX requests are coming through. All pages except Teams page are now responsive!
Try it out!

Voice-Chat

Voice-chat is an app that was incredibly difficult to program and is still a work in progress. It utilizes WebRTC and Socket.io to negotiate a peer-to-peer connection between two broswers and pass their own audio streams to each other. It also has a real-time chat function and user-status updates.

Right now, the only thing that doesn't work is the audio stream transfer. Connections are being negotiated, but the final step of the connection which involves the audio streams being passed is not functioning. I have the local audio stream being routed back into the users own audio-out, so make sure you have headphones on or you block microphone access when you load the page!

EvitOrg

EvitOrg is an event organization app created by Erika and I. It offers the ability to create tasks for an event that you are hosting and assign those tasks to an attendee. Others can also volunteer for a task if they are already attending. It is a Rails app that includes BCrypt, Paperclip, and Bootstrap.

The database is SQlite3, and there is extensive jQuery that was coded by me to make the transitions from page to page look more interesting. In the future, we would like to add custom event-page styling and push notifications when a new event is created by a friend or a task is assigned to you.

Image upload is not supported on heroku, as the remote database will not allow images to be uploaded to conserve space.
