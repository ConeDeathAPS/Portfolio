Formula1 DataBucket

This project was inspired by my passion for Formula 1. It uses get requests to the Ergast API to retrieve fastest laps for tracks and Driver/Team information. Please be patient as the API is very slow to respond at times :-P

The only page reloading that happens is when you click on a nav button at the top. Everything else is done via AJAX.

The graphs that are displayed on the driver and team pages are created by a custom graphing library that I built from scratch.
